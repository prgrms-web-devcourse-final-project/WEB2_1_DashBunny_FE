## 서비스 소개 및 기획의도

### 개요

**Dash Bunny**는 친근한 배달원 캐릭터 토끼를 활용해 빠르고 효율적인 딜리버리 서비스를 제공하는 프로젝트입니다.기존 배달 앱의 한계점인 메뉴 큐레이션을 쇼츠(Shorts)콘텐츠를 통해 제공하여 사용자 경험을 증진하는 것을 목표로 하는 배달 플랫폼 입니다.
